# Fitcell Gym - Official Website 🏋️‍♂️

A modern, fully responsive, single-page landing website built for **Fitcell Gym**, a premier fitness centre located in Aligarh. The site features a striking dark-theme design with gold accents, aimed at showcasing the facility, expert trainers, real member transformations, and membership plans.

**Live Demo:** [https://fitcellgym.vercel.app](https://fitcellgym.vercel.app)

---

## ✨ Features

* **Dark-Theme UI:** High-contrast, premium aesthetic using pure CSS variables for easy theme management.
* **Fully Responsive:** Fluid layouts built with CSS Grid and Flexbox, ensuring perfect viewing on desktops, tablets, and mobile devices.
* **Sticky Navigation:** Blurred backdrop navigation bar that transitions smoothly on scroll, including a mobile-friendly hamburger menu.
* **Infinite CSS Ticker:** A seamlessly looping marquee highlighting key gym services.
* **Working Contact Form:** Integrated endpoint (via Formspree) to send user inquiries directly to the gym's email without needing a backend.
* **Custom Map Integration:** Google Maps embed stylized with CSS filters to perfectly match the site's dark aesthetic.
* **Smooth Animations:** Scroll-reveal animations using the `IntersectionObserver` API.

---

## 🛠️ Technologies Used

* **HTML5:** Semantic markup for accessibility and SEO.
* **CSS3:** Custom properties (variables), Grid, Flexbox, keyframe animations, and media queries.
* **JavaScript (Vanilla):** Mobile menu toggling, smooth scrolling logic, and scroll-reveal interactions.
* **Google Fonts:** Bebas Neue, Barlow, and Barlow Condensed.
* **Hosting:** Deployed via Vercel.

## 🚀 Local Setup & Installation

Since this project is built with static files, running it locally is incredibly simple. No package managers or build tools are required.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/fitcell-gym.git](https://github.com/your-username/fitcell-gym.git)
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd fitcell-gym
    ```
3.  **Run the project:**
    Simply open the `index.html` file in your preferred web browser, or use an extension like VS Code's **Live Server** for hot-reloading.
